Процессор Intel Core i3-6100 3.7GHz/8GT/s/3MB (BX80662I36100) s1151 BOX

Новый процессор Intel Core i3-6100 6-го поколения, с кодовым названием микроархитектуры Skylake. Предназначен для настольной платформы Intel LGA 1151. Intel Core i3-6100 производится по стандарту 14-нм техпроцесса, имеет 2 ядра, которые работают в 4 потока на штатной тактовой частоте 3.7 ГГц. Объем кэш-памяти 3 уровня равен 3 МБ. Имеет 2-х канальный контроллер памяти DDR4 / DDR3L.

Характеристики

  • Intel Core i3
  • Socket 1151
  • Количество ядер:2
  • Intel HD Graphics 530
  • Поколение процессора Intel-Skylake (шестое)
  • Частота шины данных: 8 ГТ/с
  • Внутренняя тактовая частота: 3700 МГц
